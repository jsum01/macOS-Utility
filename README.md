# macOS 유틸리티

## 패키지 매니저
### 1. Homebrew
1-1. brew 설치하기
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
```bash
 brew install wget
```

### 2. Homebrew Cask
2-1 cask 설치하기
```bash
brew install cask
```
2-2 brew cask로 Chrome 설치하기
```
brew install --cask google-chrome
```
